# AI assets (Claude / prompts)

This folder holds **reusable prompts** and **skills** for AI-assisted work on this repo.

| Path | Purpose |
|------|---------|
| `prompts/` | One-shot or repeatable generation prompts (e.g. backend scaffold). Copy into your AI session as the user message. |
| `skills/` | Claude-style skill files or project-specific skill snippets you maintain alongside the codebase. |

Project-wide instructions for Claude Code live at the repo root in **`CLAUDE.md`** (next to **`README.md`**).
